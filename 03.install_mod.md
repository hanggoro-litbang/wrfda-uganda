# Model Installation
## Export Variables
```console
export prefix=/home/$USER/wrflibs
export WRF_EM_CORE=1
export NETCDF=$prefix
export NETCDF4=1
export HDF5=$prefix
export jasper=$prefix
export JASPERLIB=$prefix/lib64
export JASPERINC=$prefix/include
export WRF_DA_CORE=0
export WRFIO_NCD_LARGE_FILE_SUPPORT=1
```
## Download WRF, WPS
```console
cd ~
mkdir -p WRF
cd ~/raw
wget https://github.com/wrf-model/WRF/releases/download/v4.7.0/v4.7.0.tar.gz
tar -zxvf v4.7.0.tar.gz
mv WRFV4.7.0 ~/WRF
```
```console
cd ~/raw
wget https://github.com/wrf-model/WPS/archive/refs/tags/v4.6.0.tar.gz
tar -zxvf v4.6.0.tar.gz
mv WPS-4.6.0.tar.gz ~/WRF
```
## Or copy from path below
```console
cd ~/WRF
cp /mnt/VolA/training-wrfda/rawWRF/* .
```
### 01. Install WRF
```console
cd ~/WRF
LD_PRELOAD=/lib64/libm.so.6 tar -zxvf WRF-4.7.0.tar.gz
cd ~/WRF/WRFV4.7.0
./configure
```
select 78 for Intel dmpar and 1 for basic installation
```console
./compile -j5 em_real >& compile.log &
```
to check the installation progress and you may check the error messages during installation
```console
tail -f compile.log
```
Check the output
```console
ls -ls main/*.exe
```

### 02. Install WPS
```console
export prefix=/home/$USER/wrflibs
export jasper=$prefix
export JASPERLIB=$prefix/lib64
export JASPERINC=$prefix/include
export WRF_DIR=../WRFV4.7.0
export NETCDF=$prefix
```
```console
cd ~/WRF
LD_PRELOAD=/lib64/libm.so.6 tar -zxvf WPS-4.6.0.tar.gz
cd ~/WRF/WPS-4.6.0
```
WPS-4.6.0 have a bug to resolve (https://github.com/mgduda/WPS/commit/4063e190)
```console
vi geogrid/src/output_module.F
```
edit at line 471
```console
optstatus = 0
do while (ifieldstatus == 0)
```
to
```console
optstatus = 0
is_subgrid_var = .false.
do while (ifieldstatus == 0)
```
Select 19 for Intel dmpar
```console
 ./configure
```
edit file configure.wps and add command below (line 62)
```console
COMPRESSION_LIBS    = -L/home/{user}/wrflibs/lib -ljasper -lpng -lz
```
to
```console
COMPRESSION_LIBS    = -L/home/{user}/wrflibs/lib -lpng -lz \
-L/home/{user}/wrflibs/lib64 -ljasper
```
line 67 & 68
```console
DM_FC               = mpif90
DM_CC               = mpicc
```
to
```console
DM_FC               = mpiifx
DM_CC               = mpiicx
```
to check the installation progress and you may check the error messages during installation
```console
./compile -j 5 >& compile.log &
```
```console
tail -f compile.log
```

### 03. Install WRFPLUS
```console
export prefix=/home/$USER/wrflibs
export NETCDF=$prefix
cd ~/WRF
cp -r WRFV4.7.0 WRFPLUSV4.7.0
cd  WRFPLUSV4.7.0
./clean -a
```
Choose 40 for Intel and dmpar
```console
./configure wrfplus
```
edit file configure.wrf and add command below (line 120)
```console
LIB_EXTERNAL    = \
                      -L$(WRF_SRC_ROOT_DIR)/external/io_netcdf -lwrfio_nf -L/home/{user}/wrflibs/lib -lnetcdff -lnetcdf      -L/home/{user}/wrflibs/lib -lhdf5hl_fortran -lhdf5_hl -lhdf5_fortran -lhdf5 -lm -lz
```
to
```console
LIB_EXTERNAL    = \
                      -L$(WRF_SRC_ROOT_DIR)/external/io_netcdf -lwrfio_nf -L/home/{user}/wrflibs/lib -lnetcdff -lnetcdf      -L/home/{user}/wrflibs/lib -lhdf5hl_fortran -lhdf5_hl -lhdf5_fortran -lhdf5 -lm -lz -L/home/{user}/wrflibs/lib64 -ljasper
```
and line 136
```console
DM_FC           =       mpif90 -f90=$(SFC)
DM_CC           =       mpicc -cc=$(SCC)
```
to
```console
DM_FC           =       mpiifx -f90=$(SFC)
DM_CC           =       mpiicx -cc=$(SCC)
```
```console
./compile -j 5 wrfplus >& compile.log &
```
Check the output
```console
ls -al main/wrfplus.exe
```
